<nowiki>======R</nowiki>ÉSUMÉ PARTIE I.2======

IBP a été codé en [https://php.net/ PHP 5.4.4] avec le framework [http://cakephp.org/ Cakephp 2.5.5]. 

Il s'agit d'un portage de l'application lourde IBP en version web. Le site se présente donc sous la forme d'une interface sur une seule page ; chaque modification faisant appel à une requête ajax. Elle offre aux administrateurs un accès en lecture et en écriture sur les informations concernant les étudiants et leurs référents.Ces informations sont récupérées depuis la base de données [https://www.microsoft.com/en-us/server-cloud/products/sql-server/ MSSQL] '''Pedago Epitech''' se trouvant sur le serveur '''sql-1.epitech.net'''. 

Les principaux éléments dont est constituée l'unique page d'IBP sont : 

* un tableau contenant la liste de tous les étudiants,
* un menu sur la gauche permettant de filtrer les résultats affichés dans le tableau d'étudiants,
* des onglets en dessous du tableau d'étudiants permettant de visualiser et d'éditer les informations de ces derniers.
